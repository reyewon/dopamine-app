# **App Name**: TaskFlow

## Core Features:

- Project Organization: Create and manage projects to categorize tasks. ADHD support includes a visual overview of project progress.
- Task Management: Add tasks to projects, mark them as complete. All tasks will have subtasks.
- File Upload: Upload reference files to tasks to provide detailed requirements.
- Audio Note Integration: Attach voice recordings to tasks for easy and quick note-taking. You can add multiple audio files to each task.
- Smart Task Input: Users can speak a description of their task and, using generative AI, the description and related files and voice notes will be arranged to a project and it's tasks using generative AI as a tool.

## Style Guidelines:

- Primary color: Soft peach (#FFE5B4) to maintain a friendly feel.
- Background color: Pale sand (#F8F0E3) to enhance readability and minimize distraction.
- Accent color: Muted teal (#8FBC8F) for important actions and highlights.
- Secondary Color: Light Gray (#D3D3D3) for borders and subtle dividers, providing visual structure without being distracting.
- Neutral Color: White (#FFFFFF) for primary content areas, ensuring readability and contrast.
- Body font: 'PT Sans', a versatile and friendly sans-serif.
- Headline font: 'Playfair' is used for headlines to maintain an elegant, fashionable feel; if longer text is anticipated, use 'PT Sans' for body.
- Use clear, consistent icons for tasks and categories, designed for quick recognition. Incorporate simple, visual cues for status updates (e.g., color-coded dots or progress bars) to keep information easily accessible. Prioritize a design that's intuitive and non-overwhelming.
- Design with an uncluttered layout to aid focus; projects can have progress bars for visual and understandable feedback on project performance. Employ clear, visual separations for project and tasks, as the user mentioned that shape language for visual separations is important to them. Create both Mac and phone factor apps.
